# 🌍 Tradutor Bantu
## Uma ferramenta que une pessoas através das línguas de Moçambique

---

## O que é o Tradutor Bantu?

Imagina que és médico num hospital em Nampula.  
O teu doente fala **Macua** e tu só falas **Português**.

Ou és agricultor em Tete e precisas de comunicar com alguém de Inhambane que só fala **Changana**.

O **Tradutor Bantu** resolve exactamente esse problema.

> É uma aplicação que traduz palavras, frases e conversas  
> entre as línguas de Moçambique — em tempo real.

## Que línguas fala o sistema?

O Tradutor Bantu conhece **14 línguas**:

### 🇲🇿 Línguas de Moçambique
- 🗣️ **Changana** — Sul de Moçambique
- 🗣️ **Macua** — Norte (língua mais falada do país)
- 🗣️ **Sena** — Centro (Sofala, Tete)
- 🗣️ **Ndau** — Centro (Manica, Sofala)
- 🗣️ **Lomwe** — Zambézia, Nampula
- 🗣️ **Chuabo** — Zambézia
- 🗣️ **Makonde** — Cabo Delgado
- 🗣️ **Yao** — Niassa
- 🗣️ **Português** — língua oficial

### 🌍 Línguas Internacionais
- 🌐 Inglês · Swahili · Zulu · Francês · Espanhol

## Como funciona? (simples assim)

### Passo 1 — Escolhes as línguas

```
   DE: Português        PARA: Macua
```

### Passo 2 — Falas ou escreves

Podes fazer de duas maneiras:

**Opção A — Escrever:**
```
Escreve: "Onde dói?"
```

**Opção B — Falar:**
```
🎤 Carregas no botão de microfone e falas normalmente
```

### Passo 3 — Recebes a tradução

```
✅  "Olekha oruma?"
```

E ainda podes **ouvir** a tradução em voz alta! 🔊

## Os dois modos da aplicação

### 📱 Modo Traduzir
**Para uma pessoa só.**

Útil quando precisas de traduzir uma mensagem, um documento ou uma frase.

```
┌─────────────────────────────┐
│  DE: Português              │
│  PARA: Changana             │
│                             │
│  🎤  [Botão de microfone]   │
│                             │
│  ✏️  Ou escreve aqui...      │
│                             │
│  → TRADUZIR                 │
└─────────────────────────────┘
```

---

### 🤝 Modo Conversa
**Para duas pessoas com línguas diferentes.**

Cada pessoa tem o seu painel. Falam na sua própria língua, e o sistema traduz automaticamente para o outro lado.

```
┌──────────────┐  ┌──────────────┐
│  PESSOA 1    │  │  PESSOA 2    │
│  Português   │  │  Macua       │
│              │  │              │
│  🎤 ou ✏️    │  │  🎤 ou ✏️    │
└──────────────┘  └──────────────┘

          💬 Conversa no centro:
    
    "Bom dia, como está?"
    🌐 → "Bwera odewa, muli uli?"
    
                "Nili msapho, zikomo"
            🌐 → "Estou bem, obrigado"
```

Cada mensagem traduzida **toca automaticamente em voz alta** para a outra pessoa ouvir.

## Como o sistema "sabe" traduzir?

O Tradutor Bantu usa **três fontes de conhecimento**, por esta ordem:

### 1️⃣ Base de dados própria
Frases e palavras recolhidas especificamente para línguas de Moçambique.  
É a fonte mais rigorosa — feita por pessoas que conhecem as línguas.

### 2️⃣ Inteligência Artificial especializada (Bhala.ai)
Uma ferramenta de inteligência artificial criada especificamente para línguas africanas bantu.

### 3️⃣ Google Tradutor
O Google Translate, que toda a gente conhece, é usado como apoio adicional.  
Para línguas como Macua ou Ndau que o Google não traduz directamente, o sistema usa uma **língua ponte** (por exemplo, traduz primeiro para Português e depois para Macua).

> O sistema tenta sempre a fonte mais fiável primeiro.  
> Se não encontrar na base de dados, tenta o próximo.

## Funcionalidades em destaque

✅ **Funciona no browser** — não precisa de instalar nada no telemóvel  
✅ **Voz para texto** — fala e o sistema escreve e traduz  
✅ **Texto para voz** — ouve a tradução em áudio  
✅ **Conversa em tempo real** — duas pessoas, dois idiomas, ao mesmo tempo  
✅ **Frases médicas** — vocabulário de saúde pré-carregado  
✅ **Histórico** — guarda as últimas traduções da sessão  
✅ **Funciona offline parcialmente** — traduções já feitas ficam guardadas  
✅ **Sem necessidade de conta** — não precisa de registar nem fazer login

## Impacto esperado

Moçambique tem **mais de 40 línguas bantu**.  
Mais de **60% da população** tem o Português como segunda língua.

Isso significa que milhões de moçambicanos enfrentam barreiras linguísticas todos os dias — no hospital, na escola, nos serviços públicos.

O Tradutor Bantu é uma ferramenta que:

> 🎯 **Aproxima pessoas**  
> 🎯 **Salva vidas** (comunicação médica)  
> 🎯 **Garante direitos** (acesso a serviços)  
> 🎯 **Preserva línguas** (ao digitalizá-las)  
> 🎯 **Não precisa de internet para frases guardadas**
